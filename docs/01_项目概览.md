# LinkMind 知识图谱项目概览

## 一、项目简介

LinkMind 是一个基于 Neo4j 图数据库构建的知识图谱可视化系统，旨在帮助用户构建、管理和探索个人知识库。系统采用 FastAPI + React + X6 的技术栈，提供完整的图谱 CRUD、节点管理、关系管理、知识下钻、数据导入导出等功能。

## 二、技术架构

### 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 后端框架 | FastAPI | 0.115.0 |
| 数据库 | Neo4j | 5.26.0 |
| 前端框架 | React | 18.x |
| 前端构建工具 | Vite | 5.x |
| 状态管理 | Zustand | 4.x |
| 图编辑引擎 | X6 | 2.x |
| UI 组件库 | Ant Design | 5.x |
| 容器化 | Docker | 20.x+ |

## 四、核心功能

### 1. 节点管理
- 创建、读取、更新、删除顶层节点
- 节点名称唯一性校验
- 循环依赖检测与拦截
- 三种附属展示模式（内嵌、悬浮弹窗、右侧抽屉）

### 2. 关系管理
- 创建、查询、删除节点间关联关系
- 支持关系权重和描述
- 多依赖节点管理

### 3. 知识画布
- X6图编辑引擎渲染
- 节点展开/折叠交互
- 双击下钻分层浏览
- 面包屑导航回退
- 节点搜索筛选

### 4. 笔记管理（ListItem）
- 新增、编辑、删除附属笔记
- 标签和备注拓展字段
- 批量删除功能
- 统一列表UI规范

### 5. 数据导入导出
- 完整数据JSON导出
- 本地JSON文件导入（向后兼容）
- 数据备份与恢复

### 6. 社区聚类
- Louvain社区检测算法
- 自动更新节点community_id
- 支持手动重新聚类

### 7. 搜索筛选
- 全文索引检索
- 多条件筛选（类型、时间、度数）
- 错题/高权重/近七日复盘

### 8. 图分析
- 节点邻居展开（1-3度）
- 双节点最短路径查询

## 五、数据模型

### Note 节点
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识 |
| title | String | 标题 |
| content | String | 内容 |
| hash | String | 去重哈希值 |
| type | String | 节点类型 |
| community_id | Integer | 社区ID |
| degree | Integer | 关联度数 |

| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识 |

## 七、快速开始

### 开发模式

```bash
# 1. 启动Neo4j数据库
docker run -d --name neo4j-dev \
  -p 7687:7687 -p 7474:7474 \
  -e NEO4J_AUTH=neo4j/password \
  neo4j:5.26.0

# 2. 安装后端依赖
cd backend
uv pip install -e .

# 3. 启动后端服务
cd src
python main.py

# 4. 安装前端依赖
cd ../../frontend
npm install

# 5. 启动前端服务
npm run dev
```
