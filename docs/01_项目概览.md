# LinkMind 知识图谱项目概览

## 一、项目简介

LinkMind 是一个基于 Neo4j 图数据库构建的知识图谱可视化系统，旨在帮助用户构建、管理和探索个人知识库。系统采用 FastAPI + D3.js 的技术栈，提供完整的图谱 CRUD、社区聚类、全文检索、路径查询等功能。

## 二、技术架构

### 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 后端框架 | FastAPI | 0.115.0 |
| 数据库 | Neo4j | 5.26.0 |
| 图算法 | NetworkX | 3.4.2 |
| 社区检测 | python-louvain | 0.16 |
| 前端可视化 | D3.js | 7.x |
| 容器化 | Docker | 20.x+ |

## 四、核心功能

### 1. 笔记管理
- 创建、读取、更新、删除笔记节点
- Markdown文件批量导入（支持去重）

### 2. 关系管理
- 创建、查询、删除节点间关联关系
- 支持关系权重和描述

### 3. 图谱可视化
- D3.js力导向图布局
- Canvas渲染优化大数据量
- 社区聚类配色分组

### 4. 社区聚类
- Louvain社区检测算法
- 自动更新节点community_id
- 支持手动重新聚类

### 5. 搜索筛选
- 全文索引检索
- 多条件筛选（类型、时间、度数）
- 错题/高权重/近七日复盘

### 6. 图分析
- 节点邻居展开（1-3度）
- 双节点最短路径查询

## 五、数据模型

### Note 节点
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识 |
| title | String | 标题 |
| content | String | 内容 |
| hash | String | 去重哈希值 |
| type | String | 节点类型 |
| community_id | Integer | 社区ID |
| degree | Integer | 关联度数 |
| created_at | DateTime | 创建时间 |
| updated_at | DateTime | 更新时间 |

### LINK 关系
| 字段 | 类型 | 说明 |
|------|------|------|
| id | String | 唯一标识 |
| weight | Float | 权重 |
| description | String | 描述 |
| created_at | DateTime | 创建时间 |

## 六、运行环境要求

- Python 3.12+
- Neo4j 5.26.0+
- Docker 20.10+ (容器化部署)
- Node.js (可选，前端开发)

## 七、快速开始

### 开发模式

```bash
# 1. 启动Neo4j数据库
docker run -d --name neo4j-test \
  -p 7687:7687 -p 7474:7474 \
  -e NEO4J_AUTH=neo4j/password \
  neo4j:5.26.0

# 2. 安装依赖
cd backend
uv pip install -e .

# 3. 启动开发服务器
cd src
python main.py
```

### 访问地址

- 前端页面: http://{服务器IP}:8000/
- API文档: http://{服务器IP}:8000/docs
- Neo4j控制台: http://{服务器IP}:7474

**说明**：将 `{服务器IP}` 替换为实际的服务器IP地址，例如 `http://192.168.1.100:8000/`
