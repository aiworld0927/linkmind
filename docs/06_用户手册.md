# LinkMind 用户手册

## 一、系统概述

LinkMind 是一款基于知识图谱的可视化知识管理系统，帮助用户构建、管理和探索个人知识库。通过直观的图形化界面，用户可以创建知识节点、建立关联关系、添加笔记内容，实现知识的结构化管理。

## 二、快速入门

### 2.1 访问系统

| 环境 | 地址 |
|------|------|
| 开发环境 | http://localhost:5173/ |
| 生产环境 | 根据部署配置而定 |

### 2.2 首次使用

打开系统后，您将看到一个默认的知识画布。系统会自动创建一个根节点作为起始点。

### 2.3 核心操作流程

```
新增节点 → 添加关系 → 展开节点 → 添加笔记 → 下钻浏览
```

## 三、功能操作

### 3.1 节点管理

#### 3.1.1 新增节点

1. 点击顶部工具栏的「+」按钮
2. 在弹窗中填写节点名称
3. 可选：添加依赖节点（该节点依赖的其他节点）
4. 可选：选择展示模式（默认继承全局模式）
5. 点击「确定」完成创建

#### 3.1.2 编辑节点

1. 点击节点卡片上的编辑图标（✏️）
2. 修改节点名称或依赖关系
3. 点击「确定」保存修改

#### 3.1.3 删除节点

1. 点击节点卡片上的删除图标（🗑️）
2. 在确认弹窗中点击「确定」
3. 注意：删除节点会同时删除其所有子笔记

#### 3.1.4 循环依赖检测

系统会自动检测节点间的依赖关系，当您尝试创建循环依赖时：
- 系统会弹出警告提示
- 阻止该操作并显示循环路径
- 建议调整依赖关系

### 3.2 关系管理

#### 3.2.1 添加依赖关系

1. 点击节点卡片上的编辑图标（✏️）
2. 在「依赖节点」下拉框中选择目标节点
3. 点击「确定」保存

#### 3.2.2 删除依赖关系

1. 点击节点卡片上的编辑图标（✏️）
2. 在「依赖节点」列表中点击移除按钮（×）
3. 点击「确定」保存

### 3.3 知识画布

#### 3.3.1 节点展开/折叠

- **内嵌模式**：点击节点卡片展开/折叠笔记列表
- **悬浮弹窗模式**：点击节点弹出悬浮窗口显示笔记
- **右侧抽屉模式**：点击节点打开右侧抽屉显示笔记

#### 3.3.2 双击下钻

1. 双击任意节点
2. 进入该节点的子层级
3. 面包屑导航自动更新

#### 3.3.3 面包屑导航

1. 点击面包屑中的任意层级
2. 快速回退到对应层级
3. 最后一项为当前层级，不可点击

#### 3.3.4 重置画布

点击顶部工具栏的「重置」按钮，回到根节点层级。

### 3.4 笔记管理

#### 3.4.1 新增笔记

1. 在节点的笔记列表中点击「+ 添加笔记」
2. 填写笔记内容
3. 可选：添加标签（用逗号分隔多个标签）
4. 可选：添加备注
5. 点击「确定」完成创建

#### 3.4.2 编辑笔记

1. 在笔记列表中点击编辑图标（✏️）
2. 修改笔记内容、标签或备注
3. 点击「确定」保存修改

#### 3.4.3 删除笔记

1. 在笔记列表中点击删除图标（🗑️）
2. 在确认弹窗中点击「确定」

#### 3.4.4 批量删除笔记

1. 在笔记列表顶部勾选「全选」复选框
2. 或勾选多个笔记的复选框
3. 点击「批量删除」按钮
4. 在确认弹窗中点击「确定」

#### 3.4.5 空状态提示

当节点没有笔记时，会显示空状态提示和「添加笔记」引导按钮。

### 3.5 展示模式切换

#### 3.5.1 三种模式说明

| 模式 | 图标 | 说明 | 适用场景 |
|------|------|------|----------|
| 内嵌模式 | 📦 | 笔记直接显示在节点下方 | 笔记较少时 |
| 悬浮弹窗 | 💬 | 点击节点弹出悬浮窗口 | 笔记中等时 |
| 右侧抽屉 | 📋 | 点击节点打开右侧抽屉 | 笔记较多时 |

#### 3.5.2 切换模式

1. 点击顶部工具栏的模式切换按钮
2. 选择目标模式
3. 所有节点立即切换到新的展示模式

### 3.6 搜索筛选

1. 在顶部工具栏的搜索框中输入关键词
2. 系统实时过滤匹配的节点
3. 匹配的节点会高亮显示（蓝色边框）
4. 删除关键词恢复显示所有节点

### 3.7 数据导入导出

#### 3.7.1 导出数据

1. 点击顶部工具栏的「导出」按钮
2. 系统自动生成 JSON 文件
3. 文件自动下载到本地

#### 3.7.2 导入数据

1. 点击顶部工具栏的「导入」按钮
2. 在文件选择器中选择之前导出的 JSON 文件
3. 系统弹出确认弹窗，提示会覆盖现有数据
4. 点击「确定」完成导入
5. 系统自动刷新画布显示导入的数据

#### 3.7.3 格式兼容

- 支持新旧版本备份文件的自动识别
- 导入旧版本文件时会显示兼容提示

## 五、常见问题

### 5.1 节点为什么无法添加依赖？

可能原因：
- 形成了循环依赖
- 目标节点不存在

解决方法：检查依赖关系是否会形成循环，确保目标节点已创建。

### 5.2 笔记内容丢失怎么办？

解决方法：
- 系统自动保存到浏览器 localStorage
- 刷新页面后数据会自动恢复
- 定期导出数据作为备份

### 5.3 如何恢复到之前的层级？

解决方法：
- 使用面包屑导航点击目标层级
- 点击「重置」按钮回到根节点

### 5.4 导出的文件无法导入？

解决方法：
- 确认文件是从 LinkMind 导出的 JSON 文件
- 检查文件是否被修改或损坏
- 尝试重新导出并立即导入

### 5.5 画布显示异常怎么办？

解决方法：
- 刷新页面
- 清除浏览器缓存
- 检查浏览器控制台是否有错误信息
